# 认知环自我评估

请基于当前对话内容，进行认知环节自我评估：

## 1. 当前环节判断
- 你认为自己现在处于哪个认知环节？（感知/理解/建模/求解/执行/校验/修正）
- 为什么这样判断？引用你的实际行为作为依据。

## 2. 出口条件检查
{% if phase == "perceive" %}
- 事实清单：你已收集到哪些事实？
- 信息缺口：还有哪些是不知道的？
- 信息源可靠性：你的信息来自哪里，可靠程度如何？
{% elsif phase == "understand" %}
- 本质：这个问题的本质是什么？（一句话）
- 主要矛盾：核心张力在哪里？
- 关键假设：你做了哪些假设，是否已验证？
{% elsif phase == "model" %}
- 系统层次：这个系统在哪个层次上？（物理/工程/产品/社会-技术）
- 边界：系统的边界在哪里？
- 组分和关系：系统由哪些部分构成，它们怎么关联？
{% endif %}

## 3. 推进建议
- 如果出口条件满足：建议推进到下一环节
- 如果出口条件不满足：还需要做什么才能满足？
- 如果遇到阻塞：是否需要回退到上一环节？

## 4. 深度评估
- 当前任务复杂度是否需要调整深度等级？
- 是否需要提升系统层次（如从工程系统提升到产品系统）？
